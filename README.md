# Element-Merging
An experimental game Element Merging by Lime Inc.

## Recipies:
```
1. Soil + Water = Plant
2. Plant + Water = Tree & Fire
4. Fire + Soil = Rock
5. Tree + Rock = Log
6. Log + Rock = Tools
7. Log + Tools = Plank
8. Fire + Log = Campfire
10. Tools + Rock = Iron Ore
11. Iron Ore + Tools = Raw Iron
12. Soil + Tools = Sand
13. Sand + Fire = Glass
```
